# 被动记忆英语词库
## 简介
- 中英文对照映射-被动记忆英语词库-基于双拼ABC方案的自定义短语词条库-格式(from搜狗输入法)
- 简单学英语 慢就是快 通过打字的过程 学习并强化记忆自己的中英文
- 输入中文或者英文都有双语映射在下方词条
- 这样就实现了被动记忆 一个单词可以扩展很多词条/短语搭配/甚至句子
